Project Overview

The CyBot project was an autonomous vehicle application developed for CprE 288. The robot navigated a test field with obstacles, boundaries, and holes to reach a destination zone marked by four pillars. The system integrated multiple sensors and embedded components to enable real-time decision-making and remote control capabilities.

Our team, Dune Spice, implemented a complete embedded system with user interface, sensor integration, object detection, autonomous navigation, and wireless communication between the robot and base station.

Course Context

CprE 288 - Embedded Systems International

System Architecture

The system architecture integrated multiple hardware components with software modules to enable autonomous operation:

  • Sensor Input: IR and ultrasonic sensors detected objects and measured distances at 30Hz sampling rate
  • ADC & PWM: Analog-to-digital conversion processed sensor data while pulse-width modulation controlled the servo motor
  • Processing Units: ADC Routines, Sound Routines, and GUI modules handled data processing and user interaction
  • Communication: UART and PuTTY enabled wireless communication with the base station
  • System Coordinator: Spice Collector managed system timing (30Hz Sys Tick) and coordinated between all modules
  • Additional Sensors: Cliff sensors and bumper sensors provided safety features for hole and obstacle detection
