Project Overview

A cybersecurity platform that uses machine learning to detect and visualize security threats in real-time. The system processes security events, identifies suspicious patterns using AI, and displays threats on a live dashboard.

Key Features

Processes 1,000+ events per second with sub-100ms detection latency. Uses unsupervised machine learning to detect unknown threats without requiring labeled training data.

How It Works

System Flow

  • Event Generation: Simulates security events like failed logins and privilege escalation
  • Stream Processing: Apache Kafka handles high-volume event streaming
  • ML Analysis: Isolation Forest algorithm detects anomalies in real-time
  • Live Dashboard: WebSocket updates show threats instantly

Attack Detection

The system identifies five main threat types:

  • Brute force attacks (multiple failed logins)
  • Privilege escalation attempts
  • Data exfiltration (large downloads)
  • After-hours access
  • Port scanning
